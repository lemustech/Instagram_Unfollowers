# Instagram Unfollowers Console Tool

Script en JavaScript para ejecutar directamente desde la consola del navegador que analiza tu cuenta de Instagram, detecta usuarios que no te siguen de vuelta y muestra una interfaz visual integrada para buscar, filtrar, paginar, exportar resultados y gestionar una whitelist.

## Features

- Ejecutable directamente desde la consola del navegador
- Interfaz visual integrada sobre Instagram
- Detección de usuarios que no te siguen de vuelta
- Búsqueda y filtros por distintos criterios
- Gestión de whitelist
- Exportación a JSON y CSV
- Copiado rápido de usernames al portapapeles
- Paginación, estadísticas e historial de acciones
